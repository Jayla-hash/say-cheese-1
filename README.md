Browser Support
---------------

### Snapshot Support

 - Chrome 32+ (WebRTC, requires a https connection)
 - Firefox 26+ (WebRTC)
 - Microsoft Edge 13+ (ORTC)
 - IE11, Safari and iOS devices are not supported
