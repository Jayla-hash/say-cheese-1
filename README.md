Say Cheese
==========

Take a webcam snapshot or paste image data and upload it into the WordPress Media Library.

 - Developed in WP 3.8
   Should work with WordPress 3.5+
 
 - German localization.

Browser Support
---------------
Works well in Chrome and Firefox. IE and iOS6 support planned. Safari Desktop: nope...
